## Tips


### To Run Your Code in One Line

On Linux

```bash
python3 A.py < A-sample.in > A-sample.out
g++ A.cpp && ./a < A-sample.in > A-sample.out
```

On Windows

```bash
python3 A.py < A-sample.in > A-sample.out
g++ A.cpp && a < A-sample.in > A-sample.out
```

On macOS

```bash
python3 A.py < A-sample.in > A-sample.out
g++-4.9 A.cpp && ./a < A-sample.in > A-sample.out
```
